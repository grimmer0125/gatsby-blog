### 其他

**特殊 case :**  

```
var a = 1;

(function(){
  console.log(a);    //  ? -> undefined

  var a = 100;
  console.log(a);    // 100

})();
```

因為在匿名函數獨立的 scope 內，不管 var 是放在最前面，或是最後一行，他的變數實體在該 code block 一開始就是新的了，也就是說，剛剛的 code 其實等同下面這段：

```
(function(){
  var a;
  console.log(a);  
  ..
})();
```
